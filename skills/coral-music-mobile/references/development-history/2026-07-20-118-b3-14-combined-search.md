# B3-14 综合歌曲搜索

- 阶段：Batch 3 / Phase 2

- 负责人：Codex
- 开始时间：2026-07-20
- 完成时间：2026-07-20

## 目标、范围与依赖

在单来源搜索之外提供“综合搜索”：并行请求已接入歌曲搜索的酷我、网易云和咪咕，同页展示保留来源标识的结果。

复用 `OnlineCatalogService` 与现有分页/旧响应隔离；不伪造未接入的酷狗/QQ 搜索，不跨来源合并成同一首歌，也不让一个来源失败阻断其他结果。

## 实施记录

- 2026-07-20：开始实现。综合模式沿用当前页码分别请求三来源，合并稳定顺序为酷我、网易云、咪咕；仅在所有来源均失败时向用户显示错误。
- 2026-07-20：搜索来源菜单加入“综合搜索”；单来源切换会回到原逻辑。综合结果逐项展示来源标签，播放、收藏、下载和加入列表继续直接使用原始曲目来源。
- 2026-07-20：控制器并行请求三个来源并隔离异常；若至少一个来源有响应则返回该批结果，所有来源失败才显示错误。
- 验证：`dart format`、`flutter test test/search_controller_test.dart`、`flutter analyze` 与 `git diff --check` 均通过。

## 验收与下一步
