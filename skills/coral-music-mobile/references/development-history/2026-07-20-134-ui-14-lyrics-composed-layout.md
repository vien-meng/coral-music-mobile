# UI-14 歌词页分区布局

- 阶段：高保真 UI / 播放详情
- 状态：DOING
- 负责人：Codex
- 开始时间：2026-07-20

## 范围

按设计稿把歌词页改为固定三段式：顶部显示小封面、歌名和歌手，中间区域独立滚动并自动定位当前歌词，底部固定显示播放进度、时间和上一首/播放暂停/下一首。

复用现有 `CoverImage`、播放器状态、队列切歌和 seek，不新增依赖；播放页与歌词页共享同一传输控制组件，避免两套播放逻辑漂移。

## 实施与验证

- 2026-07-20：开始实现。保留现有左右滑动、歌词逐字高亮、翻译/罗马音和重试状态，只调整歌词页的页面编排。
